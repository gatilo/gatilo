Gatilo web server
=================

A simple, robust, multithread HTTP and HTTPS server implemented using c++.


## Features

* Robust
* Multithreded
* HTTPS support


## Dependencies

* CMake (sudo apt-get install cmake)
* C++

## Compile and Run

    mkdir build
    cd build
    cmake ..
    make 
    
## Usage

See examples.cpp for example usage.
